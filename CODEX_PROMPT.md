# Primer prompt recomendado para Codex

Lee primero `AGENTS.md`, `README.md`, `data/wedding.ts`, `components/WeddingExperience.tsx` y `app/globals.css`.

Trabaja sobre la invitación de boda de Israel y Liz. Antes de cambiar código, ejecuta el proyecto y revisa la experiencia en móvil y escritorio.

Prioridades:

1. Verifica que cada escena tenga blur appear, una pausa legible y blur disappear controlados por scroll.
2. Confirma que los spans de palabras mantengan separación visual correcta en todos los tamaños.
3. Mantén la primera escena: las cabezas de infancia deben acercarse sin cubrir el texto y revelar un corazón pequeño al centro.
4. Mejora el ritmo narrativo sin volverlo cursi ni corporativo.
5. Revisa que el formulario RSVP valide, muestre estados claros y llame a `/api/rsvp`.
6. No publiques datos bancarios; la mesa de experiencias debe usar un enlace externo configurable.
7. Ejecuta `npm run lint` y `npm run build`, y reporta cualquier problema pendiente.

No reemplaces la identidad editorial por una plantilla genérica de boda.
